3. **重复交易**: 在同一波段多次触发，累计亏损放大


- V3 的"绿柱高点回调"在下跌趋势中反复触发
- 趋势过滤条件不够严格
- 没有识别出"假超跌"(实际是下跌中继)

## 💡 优化建议

### V3 需要改进的地方

#### 1. 加强趋势过滤
```python
# 当前：满足任一即可
price > MA60 OR MA20 > MA60 OR 低点抬高

# 建议：至少满足 2 个条件
if sum([price > MA60, MA20 > MA60, 低点抬高]) >= 2:
    入场
```

#### 2. 添加趋势强度指标
```python
# ADX 趋势强度
if ADX < 25:  # 震荡市
    使用 V3 逻辑
else:  # 趋势市
    只做顺势交易 (不做超跌反弹)
```

#### 3. 动态调整回调阈值
```python
# 根据波动率调整
if 波动率高：
    回调阈值 = 5%  # 提高阈值
else:
    回调阈值 = 2%  # 降低阈值
```

#### 4. 限制连续交易次数
```python
# 同一方向连续亏损 2 次后停止交易
if consecutive_losses >= 2:
    暂停交易，等待趋势明确
```

## 🎯 使用建议

### V3 适合的场景
- ✅ 震荡市 (MA20 与 MA60 缠绕)
- ✅ 波动率适中的品种 (MA、豆粕)
- ✅ 有明显高低点切换的品种

### V3 不适合的场景
- ❌ 单边下跌趋势 (IC、IF 在特定时段)
- ❌ 波动率过低的品种 (螺纹钢)
- ❌ 趋势性极强的品种

### 推荐配置
```python
# 震荡品种 (MA, 豆粕，PTA)
使用 V3 + 标准参数

# 趋势品种 (IC, IF, IH)
使用 V2 或 V3 + 严格趋势过滤

# 低波动品种 (螺纹钢)
观望或使用其他策略
```

## 📋 结论

### V3 的优势
1. ✅ 更灵敏的信号识别
2. ✅ 动态高低点判断避免重复交易
