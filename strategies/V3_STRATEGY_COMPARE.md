# TrendReversalV3 策略优化说明

## 🎯 V3 核心优化：基于 MACD 柱状图的动态高低点判断

### 问题背景 (V2 的缺陷)

V2 版本使用固定的 **30 根 K 线高点** 作为回调基准，存在问题：

```
下跌趋势中：
K 线 1-30: 高点 A (5000 点)
  ↓ 回调到 4850 (-3%) → 触发入场 ❌
  ↓ 继续下跌到 4700
K 线 31-60: 高点还是 A (5000 点)
  ↓ 回调到 4850 (-3%) → 再次触发入场 ❌ (同一基准重复交易)
  ↓ 继续下跌到 4500
```

**问题**: 在同一个下跌波段中，反复根据同一个高点基准触发交易，导致连续亏损。

---

## ✅ V3 解决方案

### 核心思路

根据 **MACD 柱状图颜色** 动态判断高低点：

- **🟢 绿柱期间 (histogram > 0)**: 标记为"上涨段"，记录期间高点
- **🔴 红柱期间 (histogram < 0)**: 标记为"下跌段"，记录期间低点

### 判断逻辑

```python
# 回溯 20 根 K 线
lookback = 20

# 绿柱期间的高点
green_high = max(收盘价 [绿柱期间])

# 红柱期间的低点
red_low = min(收盘价 [红柱期间])

# 超跌判断 (满足任一即可)
1. 从绿柱高点回调 > 2%
2. 接近红柱低点 ±1-2% (超跌反弹)
3. 从 30K 线高点回调 > 3% (传统方法，作为备选)
```

## 📊 V2 vs V3 对比

| 特性 | V2 | V3 |
|------|-----|-----|
| **高低点基准** | 固定 30 根 K 线 | MACD 柱状图颜色动态判断 |
| **重复交易风险** | 高 (同一基准反复触发) | 低 (动态基准) |
| **判断维度** | 单一 (价格回调) | 多重 (绿柱高点 + 红柱低点 + 传统) |
| **MACD 背离** | 简单 (MACD 上升) | 严格 (4 种背离模式) |
| **信号质量** | 一般 | 更高 |

## 🔍 V3 超跌判断详解

### 1️⃣ 绿柱高点回调

```
MACD 柱状图：++++++0000----
              ↑绿柱    ↑红柱
K 线价格：    5000→4900→4800
              
绿柱高点 = 5000
当前价格 = 4800
回调 = (5000-4800)/5000 = 4% ✅ 触发
```

### 2️⃣ 接近红柱低点

```
MACD 柱状图：0000----++++
              ↑红柱    ↑绿柱
K 线价格：    4800→4700→4750
              
红柱低点 = 4700
当前价格 = 4750
距离 = (4750-4700)/4700 = +1.1% ✅ 触发 (超跌反弹)
```

### 3️⃣ 传统 30K 线高点 (备选)

```
近 30 根 K 线高点 = 5000
当前价格 = 4840
回调 = (5000-4840)/5000 = 3.2% ✅ 触发
```

## 📈 V3 MACD 背离优化

### V2 (简单)
- 仅检查：MACD 当前值 > 5 根 K 线前

### V3 (严格，4 种模式)
1. **MACD 上升**: 当前值 > 5 根 K 线前
2. **柱转绿**: 柱状图由负转正
3. **柱低点抬高**: 近 10 根最低点 > 前 10 根最低点
4. **经典底背离**: 价格创新低但 MACD 未创新低

**满足任一即可触发**

## 💡 使用建议

### 适合场景
- ✅ 震荡市 (绿柱/红柱交替频繁)
- ✅ 趋势反转初期
- ✅ 波动率适中的品种

### 不适合场景
- ❌ 单边暴涨暴跌 (MACD 持续同色)
- ❌ 低波动率品种 (柱状图无明显变化)

### 进一步优化方向
1. 动态调整 `macd_hist_lookback` (根据品种波动率)
2. 结合成交量确认 (绿柱高点放量更可靠)
3. 添加时间过滤 (避免夜盘流动性不足时段)
